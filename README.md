# DAMAGE-Packages

Community package registry for the [DAMAGE programming language](https://github.com/Ultiminium/DAMAGE).

Packages are written in pure DAMAGE and auto-load on every program run after install.

---

## Installing a package

```bash
damage install packagename
damage list
damage uninstall packagename
```

---

## Submitting a package

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full guide.

The short version: fork this repo, add your package folder with a `.damg` file, open a pull request.
